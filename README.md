# Projeto 66 - Rede IoT para Avaliação de Força e Técnica em Boxe

## Descrição

Este repositório contém todos os artefactos desenvolvidos no âmbito do Projeto 22.

O objetivo do projeto consistiu no desenvolvimento da infraestrutura de uma rede IoT capaz de suportar a aquisição, transmissão, processamento e armazenamento de dados provenientes de sensores instalados em luvas de boxe, permitindo a sua posterior visualização na aplicação Unity desenvolvida no âmbito do projeto global.

## Estrutura do Repositório

_RELATORIO
Relatório final do projeto em formato PDF.

Planeamento do projeto, cronogramas e documentação inicial.

Levantamento de requisitos, análise da solução e estudo das tecnologias utilizadas.

Arquitetura da solução, diagramas, esquemas eletrónicos e documentação técnica.

03_Implementacao
Código-fonte desenvolvido durante o projeto, incluindo firmware dos dispositivos ESP32, Gateway Central, fluxos Node-RED e restantes componentes da solução.

04_Teste

prompt_set.TXT
